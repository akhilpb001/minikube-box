# DevOps box
A vagrant project with an ubuntu box with the tools needed to do DevOps

# Tools included
* Terraform
* AWS CLI
* Docker
* Ansible
* Packer
* Minikube
* Kubectl
* Helm

# Start vagrant
To start and login into vagrant box, follow the steps below:
1. Clone the repo
2. Cd into the repo
3. Run `sh start.sh`

# Running minikube
To start minikube, run `sh /vagrant/scripts/start_minikube.sh`
